#Velib service

-----------------------------------------------------------------------------------------------------------------------------------

Ce projet permet de voir pour les stations de vélib les plus proches de [Simplon] (http://simplon.co/) et du métro Hotel de ville.

Pour chaque station on peut voir:

* Une map de la station
* Le nombre de vélos total qu'il y a à cette station
* Le nombre de vélos disponibles
* Le nombre de places disponibles pour ranger son vélo

Ce projet a été créé avec [Create React App](https://github.com/facebookincubator/create-react-app).

Pour réaliser ce projet j'ai utilisé:

* L'api de [Vélib] (https://developer.jcdecaux.com/#/opendata/vls?page=getstarted)
* React Materialize
* [React gmap] (https://github.com/MicheleBertoli/react-gmaps)
